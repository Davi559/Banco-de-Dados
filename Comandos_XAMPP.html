Comandos XAMPP para Bancos de Dados

Iniciar Apache e MySQL

No XAMPP Control Panel, clique em "Start" para "Apache" e "MySQL".

Acessar o MySQL via Terminal

mysql -u root -p

Definir Senha para o Root

mysqladmin -u root password sua_senha

Criar Banco de Dados via phpMyAdmin

Acesse http://localhost/phpmyadmin
Clique em "Databases"
Digite o nome do banco e clique em "Create"

Criar Tabela via SQL

CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100)
);

Backup de Banco de Dados

mysqldump -u root -p nome_banco > backup.sql

Restaurar Banco de Dados

mysql -u root -p nome_banco < backup.sql

Verificar Status do MySQL

mysqladmin -u root -p status

Parar Serviços

No XAMPP Control Panel, clique em "Stop" para "Apache" e "MySQL".

Configurar Arquivo my.cnf

Edite C:\xampp\mysql\bin\my.cnf para ajustar configurações do MySQL, como porta ou limite de conexões.
