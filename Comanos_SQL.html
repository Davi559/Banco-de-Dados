Comandos SQL

Criar um Banco de Dados:

CREATE DATABASE nome_banco;

Criar uma tabela:

CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100),
    data_cadastro DATE
);

Inserir dados:

INSERT INTO usuarios (nome, email, data_cadastro) 
VALUES ('João Silva', 'joao@example.com', '2025-05-09');

Consultar dados:

UPDATE usuarios 
SET email = 'novo.email@example.com' 
WHERE id = 1;

Excluir dados:

DELETE FROM usuarios WHERE id = 1;

Selecione com Condições:

SELECT * FROM usuarios 
WHERE data_cadastro > '2025-01-01' 
ORDER BY nome ASC;

Agrupar Dados:

SELECT COUNT(*) AS total_usuarios 
FROM usuarios;
